# Customer Churn Prediction

This project predicts whether a customer will churn using machine learning and deep learning models.

## Dataset
- File: `customer_churn.csv`
- Contains customer demographics, service details, and churn labels

## Models Used
- Artificial Neural Network (TensorFlow / Keras)
- XGBoost Classifier

## Notebook
- `Churn_prediction.ipynb`

## How to Run
1. Clone the repository
2. Install required libraries
3. Open the notebook and run all cells
